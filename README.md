SQL Data Warehouse Project 🏗️

This project demonstrates how to build a modern Data Warehouse using Microsoft SQL Server.  
It covers the complete process from raw data to meaningful business insights 📊.

What This Project Does 📌

- Collects raw data 📂  
- Cleans and transforms it 🔄  
- Loads it into structured warehouse tables 🗃️  
- Uses a Star Schema (Fact & Dimension tables) ⭐  
- Runs analytical queries for reporting 📈  

Project Structure 🧱

Sql_Data_Warehouse_Project/
- datasets/  → Raw data files  
- scripts/   → SQL scripts (tables, ETL, queries)  
- docs/      → Documentation  
- tests/     → Data validation queries  

Tools Used 🛠️

- Microsoft SQL Server  
- T-SQL  
- SQL Server Management Studio (SSMS)  

Process Followed 🔁

1. Load raw data into staging tables  
2. Clean and transform the data  
3. Create dimension tables  
4. Create fact tables  
5. Run analytics queries  

Example Analysis 📊

- Total sales by region  
- Monthly revenue trends  
- Top-selling products  
- Customer analysis  

How to Run 🚀

1. Clone the repository  
2. Open the scripts in SSMS  
3. Run them in order (Database → Tables → ETL → Queries)
